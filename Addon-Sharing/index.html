Step 1: Download the WhatsApp Helper(BETA) File

Click the button below to download the WhatsApp Helper add-on file to your computer.

Download WhatsApp Helper Version 1.0.0

Note: After clicking, the file whatsapp-helper(beta).xpi will save to your Downloads folder (or to your browsers default download location). If you are following these steps on firefox, After you download the file automatic installation will pop up on the top right corner of your screen, tick all check boxes and follow the instruction else follow the instructions here!

Step 2: Open Firefox Add-ons Manager

Open your Firefox browser and follow these steps:

  1. Click the menu button (three horizontal lines) in the top-right corner.
  2. Select Add-ons and Themes from the dropdown menu.

Step 3: Install the Add-on

Now, let's add WhatsApp Helper to Firefox:

  1. In the Add-ons Manager, click the gear icon (⚙️) at the top or go to the Extensions tab.
  2. Choose Install Add-on From File from the menu.
  3. A file picker will open. Find and select whatsapp-helper(beta).xpi from your Downloads folder, then click Open.
  4. Firefox will ask, "Add WhatsApp Helper?" Click Add to confirm.

Step 4: You're Done!

WhatsApp Helper(BETA) is now installed! You'll see it in your Extensions list, and it'll start working on web pages right away.

Troubleshooting

Common Issues

  • If the download doesn't start, try right-clicking the button and selecting "Save Link As…"
